# Beechtree Database Management System 👗

## Overview
Beechtree is a full-stack web-based database management system developed to simulate operations of a clothing brand. The system manages multiple product categories along with customer data, enabling efficient handling of business operations through a structured interface.

## Key Features
- Manage multiple product categories:
  - Jewelry
  - Cushions
  - Stitched Dresses
  - Unstitched Dresses
  - Arts
  - Scarves
  - Campaigns
- Customer data management
- Full CRUD operations (Create, Read, Update, Delete)
- Organized and scalable data handling

## System Architecture

### 🔹 Database Layer
- **MongoDB** used for flexible and scalable data storage
- Separate collections for each category
- Schema design for structured data management

### 🔹 Backend
- Developed using **Python (Flask)**
- RESTful API for handling requests
- Data validation and error handling implemented

### 🔹 Frontend
- Built using **HTML and CSS**
- Simple and user-friendly interface
- Forms for data entry and interaction

## Functionality
- Add new records for products and customers
- View and manage existing data
- Update records dynamically
- Delete unwanted entries
- Maintain structured and persistent data storage

## Technologies Used
- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **Database:** MongoDB  

## Project Purpose
This project was developed as part of a **Database Management Systems (DBMS)** course to demonstrate practical implementation of database design, backend integration, and full-stack development.

## How to Run
1. Install required dependencies (Python, Flask, MongoDB)
2. Start MongoDB server
3. Run the Flask application
4. Open the frontend in a browser
5. Interact with the system
